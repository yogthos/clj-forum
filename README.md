# clj-forum

## Goals

* REST API
* OAuth logins
* REPL
* syntax highlighting
* Markdown in comments
* Moderated
* Mobile friendly
* Topics: news, questions, etc.
* live chat?

## Prerequisites

You will need [Leiningen][1] 2.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
